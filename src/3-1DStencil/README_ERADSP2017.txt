Pratica 1 
1) Completar a versão simples do código (indexação, lançamento do kernel).
2) Profile: guardar tempo de execução da versão sem memória compartilhada

Pratica 2
Completar versão 2: 
1) acrescentar código para trazer os dados para a memória compartilhada 
2) processar localmente
3) copiar o resultado para a memória global
